# Proyek DevOps Pertama - Full-Stack Automation
Website dinamis berbasis PHP dan MySQL yang diotomatisasi dengan Docker & GitHub Actions.

## Fitur & Teknologi:
* **Containerization:** Docker & Docker Compose (Multi-container).
* **CI/CD:** GitHub Actions untuk build & push otomatis ke Docker Hub.
* **Infrastructure:** Linux (WSL 2) & Nginx/Apache.
* **Database:** MySQL dengan Docker Volumes (Data Persistence).
